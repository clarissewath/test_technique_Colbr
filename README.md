## Installation

Configure .env file to connect your database(db):

```bash
  cd api-test
  api-test/.env
  
```

```
  DB_CONNECTION=    //the db that you're using(mysql, pgsql...)
  DB_HOST=localhost 
  DB_PORT=     //db port
  DB_DATABASE=    //db name
  DB_USERNAME=    //your db username or root...
  DB_PASSWORD=    //db password if any
```

Start your prject with 

front :

```bash
  cd front-end
  npm run dev
```

back :

```bash
  cd api-test
  php artisan serve
```
